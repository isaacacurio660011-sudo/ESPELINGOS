El archivo ESPELINGOS.cpp, desarrollado en la rama heidy español-ingles, corresponde a un programa educativo en C++ cuyo objetivo es reforzar el aprendizaje de vocabulario básico de informática mediante la traducción de términos del español al inglés. El programa funciona como un cuestionario de opción múltiple, utilizando estructuras básicas como variables, condicionales y entrada/salida de datos para interactuar con el usuario y evaluar sus respuestas de manera sencilla y didáctica.
La versión 0.1 corresponde a la primera versión funcional del programa, donde se implementó la estructura principal, la función main(), las preguntas del cuestionario y la lógica básica para validar las respuestas del usuario.
En la versión 0.2, se realizaron mejoras al código, principalmente relacionadas con la organización y la claridad, añadiendo más comentarios para facilitar la comprensión del funcionamiento del programa.
Además, se observan confirmaciones de cambios con el mensaje “Agrega comentarios”, lo que indica que el código fue documentado progresivamente, siguiendo buenas prácticas de programación y control de versiones.
Este historial de versiones demuestra un desarrollo ordenado del proyecto y el uso adecuado de GitHub para el seguimiento y mejora del código.
